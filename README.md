# 🚀 PyInstaller GUI 自动化打包引擎 v5.9

一个基于 `ttkbootstrap` 开发的跨平台 PyInstaller 图形化界面工具。
告别繁琐的命令行参数，通过直观的 UI 界面，一键将您的 Python 脚本打包为 Windows、macOS 和 Linux 下的独立可执行文件。

## ✨ 核心特性

* **🖥️ 现代化 UI 界面**：支持深色/浅色主题一键切换，界面直观友好。
* **⚙️ 智能环境配置**：支持读取 `requirements.txt`，一键自动在后台执行 `pip install` 安装依赖。
* **💾 极简多项目管理**：
  * **自动记忆**：关闭软件自动保存当前所有配置，下次打开无缝衔接。
  * **配置分离**：支持将不同项目的打包参数导出为 `.json` 文件，随时一键导入。
* **🛠️ 高级打包参数全覆盖**：
  * 单文件模式 (`-F`) 与无控制台模式 (`-w`)
  * 自定义应用图标 (`.ico` / `.icns`) 与输出目录
  * 附加数据文件/目录 (`--add-data`)
  * 隐式导入库 (`--hidden-import`) 与 排除冗余模块 (`--exclude-module`)
  * 极限压缩 (`--upx-dir`) 与 Windows 管理员权限提权 (`--uac-admin`)
* **🌍 跨平台云端编译**：内置 GitHub Actions 工作流，推送代码即可自动编译发布三端（Win/Mac/Linux）独立软件。
